Response Timestamp & Star Rating Skill
A Claude Skill that automatically prepends every response with a timestamped header and includes a built-in star rating system for bookmarking your best ideas.
***What It Does
Timestamp — Every single Claude response starts with a date/time stamp in italics, like a journal entry:
> Sat Mar 07, 2026 — 03:46 PM
>
> Here's the information you asked about...
Star Ratings — Mark important ideas with 1–3 stars so you can easily find them later:
> Sat Mar 07, 2026 — 03:46 PM ⭐⭐⭐
>
> This breakthrough idea is worth revisiting...
| Rating | Meaning |
|--------|---------|
| ⭐ | Great idea |
| ⭐⭐ | Amazing idea |
| ⭐⭐⭐ | Top notch — must go back and review |
***Why You'll Love It
Every response is timestamped — no exceptions. Long analysis, quick "yes," casual chat, code output — everything gets a timestamp.
Your conversations become searchable journals — scan through past conversations by date and time.
Star ratings act as bookmarks — flag your best ideas so they don't get lost in long threads.
Claude proactively suggests stars — when something genuinely stands out, Claude will ask if you want to rate it.
Timezone-aware — timestamps use your local timezone (default: Eastern Time).
***How to Install
Download the SKILL.md file from this repo
In the Claude desktop app, go to Customize → Skills → Create new skill
Name it response-timestamp
Paste the contents of SKILL.md into the skill editor
Save — and every Claude response will now be timestamped!
***How to Use
Timestamps
No action needed — once installed, every response automatically gets a timestamp. Just chat normally.
Star Ratings
You can star-rate any response by telling Claude:
"Give that last response 3 stars"
"Put 2 stars on the March 10th chat"
"Star that one — 1 star"
Claude will also suggest star ratings when a conversation produces a standout idea.
***Example in Action
You: What's the best way to learn GitHub as a PM?
Claude:
*Sat Mar 07, 2026 — 09:15 PM*
The best approach is to start with the basics...
You: That idea about prototyping is amazing — give it 3 stars
Claude:
*Sat Mar 07, 2026 — 09:20 PM*
Done! Marked the 09:15 PM discussion with ⭐⭐⭐
(top notch, must review).
